What I Have Learned in ITSE 1301

In this class, I am gaining foundational skills that are crucial for web development. This course focuses on the basics of HTML, CSS, and JavaScript, which are the backbone of web pages. I have learned to structure a page with essential HTML tags such as <html>, <head>, <body>, and various semantic tags that make the content more organized and accessible.

One of the key lessons in this class has been the importance of semantic HTML tags like <header>, <nav>, <main>, <section>, and <footer>. These tags not only improve the structure of the web page but also enhance accessibility, making it easier for screen readers and search engines to understand the content. I am also learning how to use inline elements like emphasis and strong to make certain parts of text stand out, adding meaning and readability to the content.

This class has also introduced me to the importance of writing clean, well-formatted code. We use tools to validate our HTML code to ensure that it follows web standards and displays correctly across browsers. Each assignment brings new insights, and I am excited to continue building on these foundational skills, which are essential for creating user-friendly, well-structured websites.
